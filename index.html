x64 Assembly: registers

  • Registers: a, b, c, d, si, di, sp, bp, r8-r15, xmm0-xmm15
  • Register widths: 8: ah/al, 16: ax, 32: eax, 64: rax
  • Function params: rdi, rsi, rdx, rcx, r8, r9
  • Function return value in: rax, rdx, xmm0
